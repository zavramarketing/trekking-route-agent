# 🗺 trekking-route-agent

AI-агент для планирования трекинг-туров и расчёта стоимости маршрутов.

## Что делает агент

Принимает параметры тура и выдаёт два результата:

1. **Маршрут день за днём** — локации, активности, сложность, транспорт, жильё с ценами
2. **Финансовую калькуляцию** — себестоимость на человека, рекомендуемая цена, минимальная группа для целевой маржи, точка безубытка

Выход — единый Markdown-файл для внутреннего использования, который затем переводится в диджитал-формат.

---

## Структура

| Папка | Содержимое |
|---|---|
| `skills/` | Промпты агента: личность, планирование маршрута, калькулятор затрат, форматирование |
| `frameworks/` | Шаблоны: маршрут, калькулятор затрат, итоговый внутренний документ |
| `tours/` | Готовые туры — операционные артефакты |

---

## Как использовать

1. Создай **Claude Project**
2. Загрузи в Project Knowledge все файлы из `skills/` и `frameworks/`
3. Начни диалог — агент запросит параметры и выдаст готовый документ

---

## Входные параметры

| Параметр | Пример |
|---|---|
| Регион / страны | Польша + Чехия |
| Количество дней | 11 |
| Целевой размер группы | 9 человек |
| Минимальный порог | 6 человек |
| Уровень сложности | Средний |
| Стиль размещения | Средний (3–4*) |
| Город отправления | Таллин |
| Целевая маржа | 35% |

---

## Выходной документ

Единый Markdown-файл содержащий:
- Программу по дням с описанием, сложностью, транспортом и жильём
- Таблицу затрат (переменные + фиксированные)
- Рекомендуемую цену и депозит
- Что входит / не входит в стоимость
- Минимальную группу для целевой маржи
- Сценарии окупаемости
- FAQ для клиентов
