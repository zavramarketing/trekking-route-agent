# [SKILL 01] ПЛАНИРОВАНИЕ МАРШРУТА

## ПРИНЦИПЫ ПОСТРОЕНИЯ МАРШРУТА

### 1. Ритм сложности
Никогда не ставь два сложных дня подряд. Паттерн:
- Сложный день → лёгкий день → средний → и т.д.
- День 1 всегда лёгкий (приезд, заселение, акклиматизация)
- Последний день всегда лёгкий (отъезд, силы на исходе)
- Самый сложный день — в середине тура, когда группа уже втянулась

### 2. Логистика переездов
- Переезд в день с трекингом: не более 2 часов
- Переезд в «транзитный» день: до 4 часов
- Предпочтительный транспорт: автобус ≥ поезд > аренда авто
- Стоимость переезда — в местной валюте и в евро
- Если переезд длинный — планируй его в первой половине дня

### 3. Структура каждого дня

Для каждого дня указывай:
```
День N: [НАЗВАНИЕ]
Локация: [Город/регион, Страна]
Сложность: 🟢 лёгкий / 🟡 средний / 🔴 сложный
Переход: ~X часов | Y км (или «—» для дня заезда)

Программа:
— [активность 1 с деталями]
— [активность 2 с деталями]
— [изюминка дня — нестандартная деталь]
— [вечерняя рекомендация]

Транспорт к следующей точке: [способ], ~[время], ~[цена] EUR/чел
Жильё: [тип + локация], [€XXX–XXX]/ночь за номер (на всю группу)
```

### 4. Изюминка каждой локации
Не копируй из путеводителя. Давай детали которые знает только тот кто был:
- Лучшее время суток для конкретного вида
- Местная еда которую надо попробовать именно здесь
- Скрытая тропа или смотровая без толп
- Локальный бар/кафе вместо туристического

### 5. Сезонность и запасные варианты
Для каждого маршрута указывай:
- Оптимальные месяцы
- Что меняется при плохой погоде на каждом ключевом участке
- Конкретный запасной вариант (не «другой маршрут», а «вместо X идём в Y»)

## ПРАВИЛА ВЫБОРА ЖИЛЬЯ

| Стиль | Тип | Ориентир цены/ночь за номер |
|---|---|---|
| Бюджет | Хостел, гестхаус | €30–60 за двухместный |
| Средний | 3* отель | €60–120 за двухместный |
| Средний+ | 4* отель | €100–180 за двухместный |
| Премиум | Бутик-отель / 5* | €180+ за двухместный |

**Приоритеты при выборе жилья:**
1. Расположение близко к маршруту (не центр города, а у входа в парк)
2. Завтрак включён (для среднего стиля и выше)
3. Возможность хранения и сушки снаряжения
4. Парковка или остановка транспорта рядом
