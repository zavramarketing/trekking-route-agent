# [SKILL 02] РАСЧЁТ СТОИМОСТИ ТУРА

## СТРУКТУРА ЗАТРАТ

### Переменные затраты (V) — на каждого клиента

| Статья | Как считать |
|---|---|
| ✈️ Перелёты | Туда + обратно, реальные цены на даты тура (Google Flights) |
| 🚌 Транспорт | Сумма всех переездов по дням, на человека |
| 🏨 Жильё | Стоимость номеров ÷ количество клиентов в номере |
| 🍽 Еда | €25–35/чел/день (зависит от страны и стиля) |
| 🎫 Входные билеты | Сумма всех парков, канаток, аттракций по программе |

### Фиксированные затраты (F) — на гида, весь тур

| Статья | Описание |
|---|---|
| ✈️ Перелёты гида | Те же рейсы что у группы |
| 🏨 Жильё гида | Отдельный номер или место в отеле группы |
| 🍽 Еда гида | Те же нормы что у клиентов |
| 🎫 Билеты гида | Те же входные по программе |

## ФОРМУЛЫ

```
P  = цена тура с человека (€)
n  = количество клиентов
V  = переменные затраты на 1 клиента (€)
F  = фиксированные затраты гида на тур (€)
M  = целевая маржа (доли, напр. 0.35 = 35%)

──────────────────────────────────────────────
МАРЖА С ТУРА:
Margin = (P×n - F - V×n) / (P×n)

РЕКОМЕНДУЕМАЯ ЦЕНА при группе n и марже M:
P_rec = (F/n + V) / (1 - M)

МИНИМАЛЬНАЯ ГРУППА для маржи M:
n_min = F / (P×(1-M) - V)

ТОЧКА БЕЗУБЫТКА (маржа = 0%):
n_break = F / (P - V)
──────────────────────────────────────────────
```

## СТРУКТУРА ЦЕНЫ ДЛЯ КЛИЕНТА

**Что входит в стоимость тура:**
- Размещение (все ночи согласно программе)
- Завтраки в отелях
- Все переезды внутри маршрута согласно программе
- Входные билеты в нацпарки и аттракции согласно программе
- Работа гида

**Что не входит:**
- Международные перелёты
- Обеды и ужины (~€25–35/день)
- Личная travel insurance (обязательна, минимум €30,000 покрытие)
- Личные расходы
- Опциональные активности вне программы

---

## ПРАВИЛО ДЕПОЗИТА

- Депозит: **30% от цены тура**, невозвратный
- Оставшиеся 70%: за 30 дней до тура (банковский перевод) или наличными при встрече с гидом
- При отмене тура оператором из-за недобора группы: депозит возвращается полностью
- Минимальная группа для проведения тура: указывается в условиях бронирования
