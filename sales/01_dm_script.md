# СКРИПТ DM — ПЕРВЫЕ КЛИЕНТЫ
> Канал: личные сообщения (Telegram, WhatsApp, Instagram)  
> Аудитория: знакомые и знакомые знакомых  
> Тон: подруга рассказывает о своей затее, не продаёт

---

## ПРИНЦИПЫ

- Цель первого сообщения — получить ответ, не продать
- Никогда не отправлять стену текста сразу
- Персонализировать первую фразу под конкретного человека
- Не давить, не торопить — предлагать, не уговаривать
- Депозит просить только после того как человек сам скажет "хочу"

---

## СООБЩЕНИЕ 1 — ОТКРЫВАЮЩЕЕ

> *Короткое. Цель: вызвать интерес и получить "да, расскажи"*

```
Привет! Затеяла тут кое-что новое — организую небольшой 
трекинг-тур в августе, Польша + Татры + Чехия, 11 дней. 
Подумала о тебе — ты же любишь такие штуки?
```

**Варианты персонализации первой фразы:**
- *"...ты же говорила что хочешь в Европу летом"*
- *"...помню ты ходила в горы в прошлом году"*
- *"...ты же любишь активный отдых?"*

---

## СООБЩЕНИЕ 2 — КОГДА ОТВЕТИЛИ "ИНТЕРЕСНО / РАССКАЖИ"

> *Даёт суть без перегруза. Заканчивается вопросом.*

```
Маршрут: Краков → Татры → Словацкий Рай → Оломоуц → 
Моравский Бескид → Адршпах → Прага.

11 дней, 15–25 августа. Группа маленькая — 8–10 человек, 
возраст примерно 25–35. Жильё уже подобрала заранее, 
нормальные отели и гостевые дома.

Цена €1,400 с человека, перелёт отдельно.

Хочешь скину программу по дням?
```

## СООБЩЕНИЕ 3 — ПРОГРАММА ПО ДНЯМ

> *Отправлять только если попросили. Можно скинуть PDF или текст.*

```
День 1: Прилёт в Краков, заселение, вечерняя прогулка
День 2–3: Краков — Вавель, Казимеж, старый город
День 4–5: Татры — треккинг в горах, Закопане
День 6: Словацкий Рай — каньоны и лестницы
День 7: Оломоуц — чешский город которого никто не знает
День 8–9: Моравский Бескид — горные тропы
День 10: Адршпах — скальный город
День 11: Прага, вылет

Всё включено: трансферы между городами, жильё, 
организация. Не включено: перелёт, личные расходы, еда.
```

## СООБЩЕНИЕ 4 — КОГДА СПРАШИВАЮТ "А КТО ЕЩЁ ЕДЕТ?"

```
Пока собираю группу — есть несколько человек 
из моего круга, все примерно нашего возраста. 
Специально держу группу маленькой чтобы не 
превращалось в автобусный тур 😄
```

## СООБЩЕНИЕ 5 — КОГДА ГОВОРЯТ "ДОРОГО"

```
Понимаю. Если считать разбивку: жильё на 11 ночей, 
все трансферы между городами, организация — 
самостоятельно выйдет примерно столько же или дороже, 
плюс время на планирование. Здесь это уже сделано.

Если хочешь могу показать из чего складывается цена?
```

## СООБЩЕНИЕ 6 — ЗАКРЫТИЕ / БРОНИРОВАНИЕ

> *Только когда человек сам сказал "хочу" или "беру"*

```
Отлично! Чтобы зафиксировать место нужен депозит €200, 
остаток за месяц до тура. Напиши когда будешь готова — 
скину реквизиты.

Мест осталось [X].
```

- `Думает` — взял паузу

- `Отказ` — не в этот раз (не удалять — следующий тур)
