## БИЗНЕС В ОДНОМ ПРЕДЛОЖЕНИИ
Трекинг-туры и AI-инструмент для составления маршрутов — для русскоязычных в Европе, которые хотят настоящий поход без хаоса планирования.

---

## STATE 1 — ПРОБЛЕМА ✅

### ICP (Идеальный клиент)
- Русскоязычные в Европе, 25–35 лет
- 1–2 поездки в год, хотят настоящий трекинг — не турпакет
- Analysis paralysis: слишком много источников, страх ошибиться

### Подтверждённые боли
1. **Фрагментация:** TikTok → поиск рейсов → Booking → Google Maps — всё разрознено
2. **Непрозрачность группы:** возрастной диапазон не указан, группа набирается вплоть до даты поездки, жильё подтверждается в последний момент
3. **Страх переплатить:** нет уверенности что цена справедливая
4. **Логистика маршрута:** как соединить несколько городов в одну поездку с транспортом

### Конкуренты
| | Юлрисан | Китайские туры | KIUR |
|---|---|---|---|
| База | Эстония | Разные | Эстония |
| Аудитория | 30–55, русскоязычные | Широкая | 25–35, русскоязычные |
| Маршруты | Доломиты и др. | Азия, Европа | Польша, Чехия, Словакия + |
| AI-инструмент | Нет | Нет | Да |
| Прозрачность | Стандартная | Стандартная | Полная (возраст, жильё заранее) |

### Wedge (точка входа)
Польша + Чехия — **незаполненная ниша** для русскоязычных: красивые маршруты, которые конкуренты не делают.

### TAM/SAM/SOM
- Оценочно: русскоязычная диаспора в Европе ~6M человек
- Активные путешественники 25–35: ~300–500K
- SOM год 1: 3–5 туров × 8–10 человек = 24–50 клиентов

### Unfair Advantage
- Личное знание маршрутов в 12+ странах
- Незаполненная географическая ниша (Польша, Чехия)
- AI-composer как второй продукт (технологический барьер)

### ⚠️ Риски
- Барьер от копирования пока слабый — временное окно 6–12 месяцев
- Юлрисан может добавить похожие маршруты

## STATE 3 — ЮНИТ-ЭКОНОМИКА ✅

### Базовый расчёт (тур Польша→Татры→Словацкий Рай→Оломоуц→Моравский Бескид→Адршпах→Прага)

| Параметр | Значение |
|---|---|
| Длительность | 11 дней (15–25 августа 2026) |
| Цена | €1,400 / человек |
| V (переменные на чел.) | €802 (транспорт, жильё, еда) |
| F (фиксированные) | €746 (гид, организация) |
| n_min (безубыточность тура) | **7 человек** |
| Маржа при n=9 | ~€515/чел. (~37%) |
| Общая маржа тура (9 чел.) | ~€4,635 |

### Формулы
```
P_rec = V + F/n + margin%
n_min = F / (P - V)
```

### Модель масштабирования
- 3 тура в год × 9 человек = выручка ~€37,800 / год
- Burn Rate: минимальный (нет офиса, нет найма до PMF)
